# Locadora de Carros

Aplicativo simples para locação de carros com a utilização de arquivos txt.

### Prerequisitos
* **Netbeans** = https://netbeans.org/downloads/

### Como Instalar

* Baixar e instalar Netbeans
* Clonar o projeto da API no Netbeans
